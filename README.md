# Odin Calculator

This project is included in The Odin Project (TOP) curriculum. This project's objective is to use HTML, CSS, and modern JavaScript to create a functional calculator while implementing clean interaction logic and responsive UI behavior.

## Project Overview

A dynamic web application intended to be a fully functional calculator. It emphasizes using JavaScript to handle application logic, DOM events for user interaction, and state management to ensure calculations are accurate across button and keyboard input.

This project strengthened my comprehension of:

- DOM manipulation and event-driven programming
- Managing application state in vanilla JavaScript
- Handling calculator edge cases (operator chaining, decimal input, delete/clear)
- Keyboard input handling and mapping

## Features

- Perform basic arithmetic operations (+, -, \*, /)
- Support modulo/percentage operation (%)
- Handle decimal number input
- Delete input step-by-step (backspace behavior)
- Clear all calculator state instantly
- Keyboard support (numbers, operators, Enter, Backspace, Escape)
- Display error state for invalid operations (e.g., division by zero)

## Built With

- HTML5
- CSS3
- JavaScript (ES6+)
